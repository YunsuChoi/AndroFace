Androface

Android Open Accessory Development Kit (ADK) & NFC Technology for Jelly Bean dispenser and every electric parts by Arduino

Download this project as a .zip file Download this project as a tar.gz file

Project AndroFace

for Google HackFair Korea 2012

Developed by Yunsu Choi | goodbye.tigris@gmail.com

Abstract

  • Project AndroFace is combination of NFC and Android Open Accessory Development Kit Protocol by identifying NFC tag`s Specfic UID or NDEF data, sending command to Arduino board.

  • This repository includes Android code and Arduino firmware either.

Testing environment

  • Android Jelly Bean 4.1 | Samsung Nexus S & Galaxy S3 LTE(South Korea U+ variant)
  • Android Ice Cream Sandwich 4.0 | Samsung Nexus S & Galaxy S3 LTE(South Korea U+ variant)
  • Arduino IDE 0023 + Arduino Mega 2560 R2 + USB Host Shield 2.0
  • Arduino IDE 1.0.2 + Arduino Mega ADK R3

Additional infomations

  • ADK library is based on ADK 2011 which annouenced at Google I/O 2011.
  • NFC function from Michael Roland
  • Sound resource from "Portal 2" by Valve Corp.
